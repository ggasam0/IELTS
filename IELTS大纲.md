# 为什么这个版本最接近真题

优化原则：

1️⃣ 合并重复场景
例如

* 公交 / 火车 / 地铁
  → 合并为 **交通信息**

2️⃣ 保留所有高频主题

例如

* accommodation
* library
* museum

3️⃣ 保留 Listening 结构

| Section | 类型   |
| ------- | ---- |
| 1       | 生活事务 |
| 2       | 公共介绍 |
| 3       | 学生讨论 |
| 4       | 讲座   |
